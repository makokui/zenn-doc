# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

## よく使うコマンド
```powershell
# 記事の作成
$ npx zenn new:article
# 記事のURLの一部となるslugを指定して作成することもできます。
$ npx zenn new:article --slug my-awesome-article

#プレビューする
$ npx zenn preview # プレビュー開始
```
